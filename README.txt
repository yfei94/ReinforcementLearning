Run frozen-lake.py for analysis on the Frozen Lake problem
Run tic-tac-toe.py for analysis on tic-tac-toe problem

board_states.p is a pickled object file for all possible board states for tic tac toe. 
You can delete it, and the tic-tac-toe.py file will recalculate those states
